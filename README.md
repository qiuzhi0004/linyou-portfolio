# 林佑 · 静态作品集网站

这是一个纯静态站点（HTML/CSS/JS），无需构建。

## 本地预览

任选其一：

- 直接用浏览器打开 `linyou-portfolio/index.html`
- 或在仓库根目录运行一个静态服务器：

```bash
python3 -m http.server 5173
```

然后打开 `http://localhost:5173/linyou-portfolio/`

## 如何替换内容

- 首页：`linyou-portfolio/index.html`
- 简历页：`linyou-portfolio/resume.html`
- 项目详情：`linyou-portfolio/projects/*.html`
- 站点样式：`linyou-portfolio/assets/styles.css`
- 交互脚本：`linyou-portfolio/assets/script.js`
- 项目封面图：`linyou-portfolio/assets/images/*.svg`
