Mellow Cup · 官网改版与小程序体验

Brand Responsive Web WeChat Mini Program Awwwards 提名
返回作品列表

背景：代理商项目节奏快、沟通链路长。需要在有限时间内，把品牌气质转译为可落地的组件化页面与全端一致体验。

目标：提升品牌识别度与内容呈现质感，同时确保 Desktop → Mobile 的响应式一致性与可访问性。

设计方向

  • 极简 + 有加粗：用强对比标题与留白建立节奏感。
  • 统一视觉组件：按钮、卡片、图文模块保持一致的边界与间距规则。
  • 全端一致：从 Desktop 到 Mobile 同一套信息结构，针对触控调整点击区域与密度。

交付范围

  • 官网关键页面：首页、产品/菜单页、门店/活动内容页等。
  • 微信小程序：核心流程页面与关键组件样式。

结果

  • 获得 Awwwards “Site of the Day” 提名（简历记录）。
